首次设置 API 密钥

为了使用本工具，您需要提供 osu! API v2 凭证。请按照以下步骤获取：

  1. 打开您的个人设置页面: osu! 账户设置。
  2. 滚动到页面底部的 "开放授权" 部分，点击 "新的 OAuth 应用"。
  3. 输入一个您喜欢的 "应用名称" (例如: OsuViewer)，然后点击 "注册应用程序"。
  4. 复制页面上显示的 "客户端 ID" 和 "客户端密钥" 并粘贴到下方对应的输入框中。

✓ 已读取保存的 API 密钥。
